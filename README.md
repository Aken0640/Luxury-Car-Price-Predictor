\# Car Price Prediction App



A Machine Learning web application that predicts car prices based on user inputs such as brand, year, mileage, fuel type, and engine specifications.



Built using FastAPI and Scikit-learn with an interactive web interface.



\---



\## Features



\- Car price prediction using Machine Learning

\- FastAPI backend integration

\- Interactive user interface

\- Pre-trained ML model support

\- Real-time prediction results



\---



\## Technologies Used



\- Python

\- FastAPI

\- Scikit-learn

\- Pandas

\- NumPy

\- HTML/CSS



\---



\## Installation



Clone the repository:



```bash

git clone https://github.com/Aken0640/car.app.git

```



Move into the project folder:



```bash

cd car.app

```



Install dependencies:



```bash

pip install -r requirements.txt

```



Run the application:



```bash

uvicorn app:app --reload

```



Open in browser:



```text

http://127.0.0.1:8000

```
